# Ecossistema

O CMMV oferece uma variedade de plugins para complementar aplicações, fornecendo recursos adicionais e otimizados para diversas necessidades. Esses plugins expandem as capacidades do framework, integrando comunicação, armazenamento, cache e muito mais de maneira nativa e bem integrada. Abaixo está uma lista dos plugins disponíveis no momento:
